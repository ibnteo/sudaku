# Клавиатура суДАку

Клавиатура 3x3, первым нажатием выбираем одну из 8 зон, вторым нажатием выбираем одну из 8 кнопок.

В слайдовом варианте выходим из центра в одну из 8 зон, и возвращаемся в центр с одной из 8 сторон.
